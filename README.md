A simple, interactive ATM simulation application designed to model core banking operations and user transactions.

---

## 🚀 Features

* **User Authentication:** Secure login using Account Number and PIN validation.
* **Balance Inquiry:** View current account balance in real time.
* **Cash Withdrawal:** Withdraw funds with standard balance checks and error handling.
* **Cash Deposit:** Deposit money seamlessly into your account.
* **Transaction History:** Keep track of recent account activities and statement details.
